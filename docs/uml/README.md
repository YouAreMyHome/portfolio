# UML Diagrams - The Dev's Pixel Room

This folder contains UML diagrams for the project in two formats:

- Mermaid (.mmd)
- PlantUML (.puml)

## Files

- usecase-core.mmd
- usecase-core.puml
- usecase-games.mmd
- usecase-games.puml
- class-architecture.mmd
- class-architecture.puml

## Scope

- Use Case Core: portfolio interactions in the 3D room and overlays
- Use Case Games: TV mini-games lifecycle and score persistence
- Class Architecture: high-level component/module relationships

## Notes

- Labels are bilingual VI/EN (ASCII form for portability).
- Class diagram is intentionally high-level to avoid component-level clutter.
